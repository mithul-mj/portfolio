Plant Disease Detection

2024 Python / TensorFlow / Django AI/ML Project

The Challenge

Farmers often struggle to identify plant diseases early, leading to significant crop losses. Traditional methods require expert knowledge and are time-consuming.

My mission was to build an AI-powered solution that could detect plant diseases with high accuracy, making it accessible through a simple mobile application.

The Architecture

I developed a Convolutional Neural Network (CNN) using TensorFlow, trained on datasets from Kaggle to achieve 95%+ accuracy in disease classification. The model was integrated with a Django REST API for seamless backend communication.

The complete solution includes an Android application that captures plant images, sends them to the API, and displays results with auto-generated PDF reports for farmers.
